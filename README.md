## 🚀 Як запустити / How to Run

```bash
# Клонуйте репозиторій
git clone https://github.com/Hafych/fun_chat_bot.git
cd fun_chat_bot

# Встановіть залежності
pip install -r requirements.txt

# Запустіть бота
python chatbot.py
```

> 💡 На Windows: якщо не працюють емодзі, спочатку виконайте:  
> ```cmd
> chcp 65001
> ```
